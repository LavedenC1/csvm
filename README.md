# csvm
Run KasmVNC in GitHub Codespaces

## Setup
1. Create a codespace with this repo as the repo
2. Run setup.sh, and go through dialogs
3. Run setup2.sh, and go through dialogs
4. Run start.sh and selected option 3 (manually edited xstartup)
5. Forward port 8444 and make sure the protocol is https
6. Access the link and enter the credentials you setup
